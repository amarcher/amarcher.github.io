The CSS "Display:" Property

Sunday, August 17, 2014

A quick reference on common values for the "display:" property:

Display: Block

I get my own line and you better respect the height and width that's been specified for me!

Block

Display: Inline

I'll share a line with other elements if there's space for my content there and don't worry about the height or width that's been specified for me, they're not important.

Inline

Display: Inline-Block

I'll share a line with other elements if there's space for me, but you better respect the height and width that's been specified for me!

How do they play together?

Note below how the "block" element insists on getting its own line. The "inline" and "inline-block" elements are willing to share the line, but only "inline-block" gets to specify its height and width!
